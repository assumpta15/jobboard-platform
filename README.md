# JobBoard – Job Recruitment Platform

A full-stack job recruitment platform built with PHP and MySQL.

## Features
- User authentication (Login / Register)
- Employer dashboard
- Job seeker dashboard
- Post job listings
- Apply for jobs with CV upload
- Accept / reject applicants
- Dashboard statistics
- Responsive UI

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
